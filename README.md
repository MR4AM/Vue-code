## 使用花生壳平台进行内网穿透将本机项目通过中转允许外网访问
> 花生壳官网 https://hsk.oray.com/
### 内网穿透流程
- 1.注册花生壳账号获取临时的域名
- 2.购买内网穿透服务，将域名IP地址指向本机的公网ip地址
- 3.配置内网映射，获取本地以太网ipv4的物理地址，配置对应的本机服务端口
- 4.将远程访问映射到对应的局域网服务
